git -> создание файла настрйоки .git 
git add <filename> -> добавление файла на загрузку 
1 git add . -> добавление всех файлов в папке на загрузку 
3  git commit -m '<message>' -> подтверждает изменения в файлах, и окончательно 
готовит их к загрузке 
git remote addd <link name> <ssh link> -> пишется один раз после git init, добавляет ссылку
на репозиторий в конфигурацию 
git push origin <branch name> -> загрузить изменения, совершенные в git commit, на сайт 

Для загрузки новых файлов повторить шаги 
(2, 3, 5)
